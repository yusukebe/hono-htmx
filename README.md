# Todo Example

Stack:

* Sonik
* htmx
* Zod
* Cloudflare Workers
* Cloudflare D1

## Usage

Install:

```
npm install
```

Setup:

```
wrangler d1 create todo --experimental-backend
wrangler d1 execute todo --local --file=todo.sql
```

Dev:

```
npm run dev
```

Deploy:

```
npm run deploy
```
